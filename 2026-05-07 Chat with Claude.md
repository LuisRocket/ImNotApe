## GeoGuessr가 재미있는 이유 분해

GeoGuessr의 중독성은 몇 가지 메커니즘에서 나옵니다:

1. **즉각적 시각 피드백** — 풍경, 표지판, 식생, 도로 표시 등이 한눈에 들어옴
2. **다층적 단서** — 언어 → 국가 → 지역 → 도로 종류 → 정확한 위치로 좁혀가는 funnel
3. **보편적 직관 + 전문 지식의 교차** — 초보도 "여긴 사막이네" 정도는 알고, 전문가는 도로 표지판 폰트로 국가를 맞춤
4. **점수 = 거리 기반 연속 보상** — 정답 못 맞춰도 가까우면 부분 점수
5. **명확한 ground truth** — 실제 좌표라는 객관적 정답

## Financial Statement 게임으로 옮겼을 때

**잘 작동할 부분:**

- **다층적 단서 구조는 살아있음**. 매출 규모 → 산업 추정 → 마진 구조 → 시대 추정 → 회사 식별 funnel이 자연스럽게 만들어짐. 예를 들어 매출 $500B + 낮은 영업이익률 + 거대한 inventory → Walmart 추측 → 매출 성장률로 연도 추정.
- **시대적 단서가 풍부함**. SBC가 매출 대비 큰 비중 → 2010년대 후반 테크. R&D가 매출 50% 초과 → 바이오텍. Goodwill 급증 → M&A 직후. 회계 기준 변경(ASC 606, ASC 842) 자체가 연도 단서.
- **점수 체계 설계 가능**. "회사 정확히 + 연도 ±1년 = 5000점, 산업만 맞춤 = 1000점" 식으로 partial credit 설계 가능.

**작동하기 어려운 부분:**

- **시각적 즉각성 부재**. 숫자 테이블은 풍경처럼 한눈에 들어오지 않음. 최소 30초~2분 분석 시간 필요. GeoGuessr의 5초 직관 게임 같은 라운드는 어려움.
- **진입장벽이 훨씬 높음**. GeoGuessr는 "야자수 = 열대" 같은 상식이 통하지만, financial statement는 회계 기초 지식 없으면 아예 진입 불가. 타겟이 좁음.
- **차별성 문제**. 성숙한 산업의 비슷한 규모 회사들(Coca-Cola vs Pepsi의 특정 사업부, 또는 빅테크 5개)은 financial statement만으로 구분이 매우 어려움. "정답이 unique해야 한다"는 게임 디자인 조건이 깨질 수 있음.
- **연도 맞추기의 난이도 편차**. GFC 직후, COVID 직후처럼 시그널 강한 해는 쉽고, 평온한 해는 거의 불가능.

## 재미를 살리려면

루이스 님의 게임 디자인 관점에서 보면, 이건 **knowledge-as-resource 루프**가 아주 강하게 작동할 수 있는 장르예요. 매 라운드마다 플레이어가 "ASC 606이 언제부터 적용됐지?" "Tesla가 흑자 전환한 게 언제더라?" 같은 메타 지식을 축적하면서 점점 강해지는 구조.

몇 가지 디자인 제안:

- **Hint 시스템을 점수와 trade-off로**. 산업 공개 = -500점, 시가총액 범위 공개 = -1000점. 플레이어가 자기 confidence를 가격 매기게 됨.
- **Multi-year 모드**. 같은 회사의 3년치 statement를 동시에 줘서 추세로 추정하게 하면 unique하게 만들기 쉬움 + 회사의 "스토리"가 보임.
- **Daily challenge**. 모두가 같은 statement로 경쟁하면 커뮤니티성 생김 (Wordle 모델).
- **"This Statement Tells a Story" 모드**. 정답 공개 후 "이 해에 무슨 일이 있었는지" 짧은 narrative 제공. 학습 효과로 retention 확보.

## 솔직한 결론

GeoGuessr 수준의 mass appeal은 어려워요 — 진입장벽이 너무 높고 시각적 즉각성이 부족해요. 하지만 **투자/회계 커뮤니티 한정으로는 매우 강력한 niche product**가 될 수 있다고 봐요. 루이스 님의 투자밤 커뮤니티 규모와 한국 투자 콘텐츠 시장을 생각하면, 한국어 버전 + 미국 대형주 + Daily 1문제 모델로 시작하면 유입 채널이 명확합니다.

오히려 GeoGuessr보다 **Worldle + Wordle의 niche 버전들**(Globle, Tradle 등)이 더 가까운 비교 대상일 것 같아요. Tradle은 실제로 "이 나라의 수출 구조를 보고 나라 맞추기" 게임인데, 컨셉이 거의 동일하면서 작은 충성도 높은 유저층을 보유하고 있어요. 한 번 살펴보시면 디자인 참고가 될 거예요.
